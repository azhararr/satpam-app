📦 ISI PACKAGE:
   • index.html        → Halaman utama
   • style.css         → Styling
   • app.js            → Logic utama (camera, GPS, watermark)
   • manifest.json     → PWA config
   • sw.js             → Service worker (offline)
   • icons/            → App icons (192px & 512px)
   • README.md         → Dokumentasi lengkap
   • DEPLOYMENT.md     → Panduan deploy detail
   • QUICKSTART.md     → Panduan cepat

🚀 CARA DEPLOY (PILIH SALAH SATU):

   1️⃣  GITHUB PAGES (Recommended):
      - Buat repository di github.com
      - Upload semua file
      - Enable GitHub Pages di Settings
      - Dapatkan link: https://username.github.io/satpam-app
      - Waktu: 5-10 menit
      → Baca: DEPLOYMENT.md

   2️⃣  NETLIFY (Paling Mudah):
      - Login netlify.com
      - Drag & drop folder ini
      - Dapatkan link otomatis
      - Waktu: 2-3 menit
      → Baca: QUICKSTART.md

   3️⃣  VERCEL:
      - Login vercel.com
      - Import project
      - Deploy
      - Waktu: 3-5 menit

⚡ FITUR UTAMA:

   ✓ Camera Only (tidak bisa upload foto lama)
   ✓ GPS Otomatis + Nama Lokasi (OpenStreetMap)
   ✓ Watermark Otomatis:
     • Nama sekolah
     • Tanggal & waktu
     • Lokasi + GPS coordinates
     • Nama satpam
     • ID laporan unik
   ✓ Share langsung ke WhatsApp
   ✓ Offline Ready (kecuali geocoding)
   ✓ Install as App di home screen

🔒 KEAMANAN:

   ✓ Hanya capture, tidak bisa upload
   ✓ GPS & timestamp real-time
   ✓ Watermark embedded (tidak bisa dihapus)
   ✓ ID unik setiap foto
   ✓ No server, no database (privacy)

📱 CARA INSTALL DI HP SATPAM:

   1. Buka link PWA di Chrome/Safari
   2. Menu (⋮) → "Add to Home Screen"
   3. Icon muncul di home screen
   4. Buka seperti app biasa

🎯 CARA PAKAI:

   1. Pilih nama satpam
   2. Klik "Ambil Foto"
   3. Capture
   4. Kirim ke WhatsApp

📝 KUSTOMISASI:

   • Ganti nama satpam: edit index.html
   • Ganti warna tema: edit style.css
   • Ganti nama sekolah: edit app.js
   → Detail di README.md

🆘 BUTUH BANTUAN?

   2. Baca DEPLOYMENT.md (deploy detail)
   3. Baca README.md (dokumentasi lengkap)
